PGMI Church Management PortalA lightweight, mobile-responsive Progressive Web App (PWA) for Precious God Ministry International. This portal handles membership management, financial recording, and soul-winning tracking.FeaturesMembership Database: Add, edit, and view member profiles with digital ID cards.Financial Management: Record Tithes, Welfare, and Expenses with receipt generation.Growth Track: "Ladder of Life" system to track new souls from winning to sending.Offline Capable: Uses local storage for offline access and syncs when online.Data Sync: Connects to Google Sheets via Google Apps Script for backend storage.Setup InstructionsNo Installation Required: This is a static HTML file.To Run: Simply double-click index.html to open it in any web browser.Mobile App: Open the link on a mobile device and select "Add to Home Screen" to install it as an app.ConfigurationThe app is currently configured to connect to the following Google Apps Script backend:https://script.google.com/macros/s/AKfycbwbJkJiS6ygI2K-mBbJuEV6jWccZ-MSdjHcV0cp46KsVTdbBYNkC3FgaFdE88y0MQvX/execTo change this, edit the GOOGLE_SCRIPT_URL constant in index.html.TechnologiesHTML5 / CSS3Tailwind CSS (via CDN)Vanilla JavaScript (ES6+)Chart.js (for dashboard analytics)Lucide Icons
